# SimpleMDB API Test - cURL Commands

## MOVIES ENDPOINTS

### List Movies (Success)
curl -X GET "http://localhost:3000/api/v1/movies?page=1&size=5"

### List Movies (Error - Invalid page)
curl -X GET "http://localhost:3000/api/v1/movies?page=0&size=5"

### Create Movie (Success)
curl -X POST "http://localhost:3000/api/v1/movies" \
  -H "Content-Type: application/json" \
  -d '{"title":"Inception","year":2010,"description":"A thief who steals corporate secrets through dream-sharing technology."}'

### Create Movie (Error - Missing title)
curl -X POST "http://localhost:3000/api/v1/movies" \
  -H "Content-Type: application/json" \
  -d '{"title":"","year":2010,"description":"Test"}'

### Read Movie (Success)
curl -X GET "http://localhost:3000/api/v1/movies/1"

### Read Movie (Error - Not found)
curl -X GET "http://localhost:3000/api/v1/movies/999"

### Update Movie (Success)
curl -X PUT "http://localhost:3000/api/v1/movies/1" \
  -H "Content-Type: application/json" \
  -d '{"title":"The Shawshank Redemption - Updated","year":1994,"description":"Updated description."}'

### Update Movie (Error - Invalid year)
curl -X PUT "http://localhost:3000/api/v1/movies/1" \
  -H "Content-Type: application/json" \
  -d '{"title":"Test","year":1500,"description":"Test"}'

### Delete Movie (Success)
curl -X DELETE "http://localhost:3000/api/v1/movies/3"

### Delete Movie (Error - Not found)
curl -X DELETE "http://localhost:3000/api/v1/movies/999"
